# Kubiyogen Frontend

Ilk gun iskeleti Next.js + Tailwind + TypeScript hedefiyle hazirlandi.

## Planlanan yapi

- `app/`: sayfalar
- `components/`: ortak arayuz parcaciklari
- `lib/`: gecici icerik ve sabitler

## Notlar

- Bu klasor sadece iskelet olarak hazirlandi.
- Lokal makinede `npm` kurulumu bozuk oldugu icin bagimliliklar henuz yuklenmedi.
- API adresi icin `.env.example` eklendi.
